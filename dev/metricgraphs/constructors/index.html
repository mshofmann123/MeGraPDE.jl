MeGraPDE.MetricGraphs.metric_barabasi_albert — Function
metric_barabasi_albert(n::Int, k::Int; ℓ=1, seed=nothing)

Create Barbási-Albert graph with 'n' vertices by growing an initial graph with 'k' vertices and attaching each vertex with 'k' edges, see Graphs.barabasi_albert.

Optional Arguments

  • ℓ: specify equilateral edge length, vector with edge length or ":non_equi" for random edge length
  • seed=nothing: set the RNG seed.
